# ESP Predictive Diagnostics (Math Engine)

Алгоритм предиктивной диагностики погружных электродвигателей (УЭЦН) на основе нейронных сетей LSTM.

## Описание
Данный модуль содержит вычислительное ядро для анализа телеметрических данных и прогнозирования остаточного ресурса изоляции обмоток ПЭД и износа рабочих органов насоса.

## Основные функции
- Математическое моделирование деградации изоляции.
- Классификация дефектов (подшипник качения / износ рабочих органов).
- Расчет прогнозных значений для предотвращения аварийных отказов.

## Технологический стек
- Python
- TensorFlow/Keras (LSTM)
- NumPy
